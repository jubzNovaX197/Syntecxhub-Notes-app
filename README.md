## рџ“– About the Project

Syntecxhub Notes App helps users create, organize, and manage notes directly in their browser.

The application supports different note types, note categorization, pinning important notes, dark mode, search functionality, and local storage persistence.

All notes are stored in the browser using **localStorage**, so data remains available even after refreshing the page.
